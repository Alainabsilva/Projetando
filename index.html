Comunicação Antirracista

Aprenda a se comunicar de forma respeitosa e inclusiva com mulheres negras. Pequenas mudanças nas palavras fazem grandes diferenças nas relações.

Racismo cotidiano em frases “comuns”

Clique nos cards abaixo para descobrir alternativas respeitosas para expressões que devem ser evitadas.

“Você é exótica.”

Racismo que se disfarça de elogio

Clique para virar

Por que fere

Sugere que a mulher negra é “fora do padrão”, reforçando que o “normal” é branco.

Como substituir

“Você tem uma beleza marcante.”

“Seu cabelo é duro.”

Estética negra como ofensa

Clique para virar

Por que fere

Trata a textura natural do cabelo crespo como algo ruim ou indesejado.

Como substituir

“Seu cabelo está cheio de personalidade.”

“Você é bonita para uma negra.”

Elogio comparativo e racista

Clique para virar

Por que fere

Coloca a beleza negra como exceção.

Como substituir

“Você é linda.”

“Você fala bem para uma pessoa negra.”

Surpresa com capacidade intelectual

Clique para virar

Por que fere

Reforça estereótipos que associam pessoas negras à ignorância.

Como substituir

“Sua forma de se expressar é admirável.”

“Você é tão inteligente... nem parece negra.”

Surpresa com capacidade intelectual

Clique para virar

Por que fere

Supõe que inteligência não é característica de pessoas negras.

Como substituir

“Sua inteligência é admirável.”

“Posso tocar no seu cabelo?”

Fetichização do corpo negro

Clique para virar

Como substituir

“Seu cabelo está lindo!”

“Preto é raça ruim.”
